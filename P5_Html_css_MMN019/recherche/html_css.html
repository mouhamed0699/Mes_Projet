Connaissances générales

Fonctionnement d’un site Web

  • Les outils nécessaires pour faire fonctionner un site web

    Les outils nécessaires pour faire fonctionner un site Web incluent un éditeur de texte, un navigateur Web et un serveur Web.

  • Les langages nécessaires pour faire fonctionner un site web

    Les langages nécessaires pour faire fonctionner un site Web incluent HTML pour la structure et le contenu, CSS pour la mise en page et le style, et JavaScript pour l'interactivité.

Balise et attribut

  • Balise

    Une balise est une étiquette qui définit la structure et le contenu d'un élément HTML.

  • Balise inline vs block
    • Les balises inline sont utilisées pour les éléments qui s'intègrent dans une ligne de texte, tels que les liens hypertextes, les images, les éléments de texte en ligne, les boutons de formulaire et les éléments de liste. Les balises inline n'ont pas de largeur ou de hauteur spécifiées et ne peuvent pas être divisées sur plusieurs lignes.
    • Les balises block sont utilisées pour les éléments qui occupent toute la largeur disponible et sont affichés sur une ligne séparée, tels que les paragraphes, les en-têtes de section, les listes et les divs. Les balises block ont une largeur et une hauteur spécifiées et peuvent être divisées sur plusieurs lignes.

    Il est important de comprendre la différence entre les balises inline et block pour pouvoir utiliser la bonne balise pour la bonne tâche. Par exemple, si vous voulez afficher un paragraphe de texte, vous devez utiliser la balise de paragraphe

    , qui est une balise block, alors que si vous voulez afficher un lien hypertexte, vous devez utiliser la balise de lien , qui est une balise inline.

Les attributs sont des éléments qui se trouvent à l'intérieur de la balise et qui fournissent des informations sur l'élément lui-même. Structure par défaut d’une page HTML

La structure par défaut d'une page HTML est généralement composée de plusieurs éléments qui sont nécessaires pour que le navigateur web puisse afficher le contenu correctement.
Voici une structure de base typique pour une page HTML :

La balise 

 <!DOCTYPE html>

déclare la version du HTML utilisée par la page.

La balise 

 <html>

est la balise racine de la page HTML.

La balise

 <head>

contient des informations sur la page, telles que le titre, la description, les balises de style CSS, les scripts JavaScript, etc.

La balise 

 <meta>

spécifie les métadonnées de la page, telles que l'encodage de caractères, la description de la page, les mots-clés, etc.

La balise

  <title>

définit le titre de la page qui s'affiche dans l'onglet du navigateur.

La balise

  <body>

contient le contenu de la page, tel que le texte, les images, les vidéos, les liens, etc.

Liens

Les liens sont un élément clé de la création de pages web. Ils permettent aux utilisateurs de naviguer entre différentes pages web, sections d'une même page ou de télécharger des fichiers.

  • Lien Externe

    Il pointe vers une ressource en dehors du site web en cours. Il est généralement utilisé pour créer des liens vers d'autres sites web ou pour lier des fichiers tels que des documents PDF, des images, etc.

  • Lien Interne

    Il pointe vers une ressource à l'intérieur du même site web. Il est généralement utilisé pour créer des liens entre différentes pages web d'un même site

    • Relatif

      C'est un type de lien interne qui utilise une adresse relative pour pointer vers une ressource. L'adresse est relative à la position de la page courante dans la structure de fichiers du site web

    • Absolu

      C'est un type de lien interne qui utilise une adresse complète pour pointer vers une ressource. L'adresse est généralement composée d'un protocole (HTTP, HTTPS), d'un nom de domaine et d'un chemin d'accès complet à la ressource.

  • Ancre

    C'est un type de lien interne qui permet de pointer vers une section spécifique d'une même page web. Les ancres sont souvent utilisées pour permettre aux utilisateurs de naviguer facilement à travers de longues pages web

  • Infobulle

    C'est une information contextuelle qui apparaît lorsque l'utilisateur survole le lien avec sa souris. Les infobulles sont souvent utilisées pour fournir des informations supplémentaires sur le lien ou la ressource vers laquelle il pointe.

  • Lien de téléchargement

    C'est un lien qui permet de télécharger un fichier à partir du serveur web. Les liens de téléchargement peuvent pointer vers différents types de fichiers tels que des documents, des images, des fichiers audio et vidéo, etc.

Comment on peut valider un document html ?

La validation d'un document HTML est un processus qui consiste à vérifier si le code HTML d'une page web est correctement structuré et ne comporte pas d'erreurs. Pour valider un document HTML, il existe des outils en ligne gratuits tels que le validateur de W3C.
